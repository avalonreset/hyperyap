HyperYap

Private desktop dictation powered by NVIDIA Parakeet. Speak, transcribe locally, and paste into the app you are already using.

Download latest release View source
No cloud transcription Speech recognition runs on your machine after setup.
English-first model choice Defaults to NVIDIA Parakeet TDT 0.6B v2 via smcleod INT8 ONNX.
Fast daily workflow Record, transcribe, and paste into editors, chats, terminals, and tools.

Built for daily dictation

Local voice input that stays out of your way.

Record and paste

Use toggle-to-talk or push-to-talk shortcuts, then paste the transcript into the active window automatically.

Private by default

Audio stays local. HyperYap collects no data and does not need the internet after the speech model is installed.

Configurable workflow

Tune shortcuts, overlay behavior, sound feedback, dictionaries, formatting rules, and optional LLM cleanup.
