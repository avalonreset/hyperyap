| `1.8.0`       | 2026-03-28 | **✨ New Features**<br>- **(Major) Voice Mode** — Activate Murmure by voice with customizable wake words, no keyboard needed. Includes auto-send enter, and wake word listener stays active during keyboard-triggered recording.<br>- **(Major) LLM Remote Servers** — Connect to remote OpenAI-compatible API servers for LLM Connect, with per-mode provider switching, model search, per-mode voice triggers, and onboarding flow.<br>- **(Major) Import/Export Settings** — Import and export configuration settings partially or fully. Add CLI import support (`murmure import <file>`) for sysadmin mass deployment.<br>- **Shortcuts** — Add support for mouse buttons, OEM keys (-, =, [, ], ;, ', etc.), <>, ², F13-F24. Add a shortcut in overlay to cancel recording.<br>- **Rules** — Add regex support for custom rules, improve rule labels, add "?" helper for replacement text, short text correction (auto-lowercase/remove punctuation), and rule reordering.<br>- **LLM** — Allow LLM mode reordering. Add "Voice Dictation" preset for interpreting spoken formatting commands as typographic characters.<br>- **Dictionary** — Add option to clear all dictionary entries.<br>- **About Page** — Redesigned with starry background and sponsor cards.<br>- **Autostart** — Start minimized to tray on autostart.<br>- **macOS** — Add configurable option to show/hide application in Dock.<br>- **Linux** — Add `.deb` package and `install.sh` one-liner script for Debian/Ubuntu.<br>- **Update** — Update `tauri-plugin-updater` to 2.10.0 for multi-format support (.deb, .exe, macOS Intel).<br>- **Sidebar** — Add "Release notes" link in sidebar footer.<br>- **Defaults** — Improve default settings for new users (transcription in RAM, trailing space).<br>**🐛 Bug Fixes**<br>- Fix overlay scrollbar on Windows, overlay stealing focus, overlay freeze on audio timeout.<br>- Fix double percentage display during download.<br>- Use dark mode colors consistently.<br>- Improve microphone list and keep manually selected device when disconnected. Fix crash when no microphone is available.<br>- Improve LLM accuracy with system prompts. Improve response speed of reasoning models by disabling thinking mode.<br>- Improve smart mode auto-spacing and word deduplication (parakeet fix).<br>- Fix selected text detection when clipboard already contains the same content.<br>- Fix ToggleToTalk accidental restart caused by X11 auto-repeat.<br>- Fix spurious modifier key release on macOS after paste and recording lock management.<br>- Fix wake word language-aware validation, history update, and EMA smoothing for VAD. |
