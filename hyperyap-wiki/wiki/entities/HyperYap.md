# HyperYap

HyperYap is a local-first speech-to-text app. Its architecture combines a Tauri Rust backend, a React TypeScript frontend, and a separate hotkey companion for global input workflows.

## Key Properties

- Runs transcription locally.
- Uses a privacy-first persistence boundary.
- Supports push-to-talk workflows.
- Supports terminal smart screenshot paste workflows.
