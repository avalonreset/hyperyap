# Privacy First Local Processing

HyperYap's core product constraint is that processing stays local and user data is not collected. Project rules allow persistence only for the last five transcriptions.

## Consequences

- Avoid telemetry and remote processing.
- Treat clipboard and screenshot content as sensitive.
- Keep release notes and vault notes focused on behavior, not private user content.
